# Anlisis-de-Datos-de-Transporte-Ferroviario-Renfe-con-Power-BI
Desarrollo de un dashboard interactivo para analizar sus datos. El objetivo es identificar patrones en los viajes, analizar la estructura de precios y entender los factores que influyen en la demanda y el coste de los trayectos.

Algunos insights interesantes:
• Los viernes concentran el mayor volumen de viajes, mostrando un patrón claro de demanda semanal.
• Las tarifas promocionales dominan el volumen, pero reducen el ingreso medio por billete.
• Los trenes AVE presentan precios medios más elevados que otras opciones como AVLO.
• Existe una alta variabilidad de precios según clase, siendo Preferente significativamente más cara.
• Las rutas entre Madrid y Barcelona concentran gran parte del tráfico.

Recomendaciones de negocio
• Ajustar precios en días de alta demanda (especialmente viernes)
• Optimizar el uso de tarifas promocionales para maximizar ingresos
• Revisar la estrategia de precios por clase para mejorar la segmentación de clientes
• Analizar rutas principales para mejorar eficiencia operativa
